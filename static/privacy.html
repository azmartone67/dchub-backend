Privacy Policy

Last Updated: January 31, 2026

Overview

DC Hub Nexus ("we", "our", or "us") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, and safeguard information when you use our data center intelligence platform and API services.

Information We Collect

API Usage Data

When you access our API endpoints, we collect:

  • API endpoint accessed and request parameters
  • Timestamp of each request
  • IP address and user agent string
  • API key identifier (for authenticated requests)
  • Response status codes and latency metrics

AI Platform Tracking

We monitor when AI platforms (ChatGPT, Claude, Gemini, Perplexity, etc.) access our data to:

  • Track platform identification via user-agent headers
  • Log queries and data retrieved for citation tracking
  • Measure usage patterns to improve our AI interconnection services

Account Information

If you register for an API key or subscribe to alerts:

  • Email address
  • Organization name (optional)
  • Webhook endpoints (if configured)

How We Use Your Information

We use collected data exclusively for:

  • Providing and improving our data center intelligence services
  • Rate limiting and abuse prevention
  • Generating aggregated, anonymized usage statistics
  • Delivering subscribed alerts and notifications
  • Tracking AI platform citations and usage

Data We Do NOT Collect

  • Personal identification beyond email (no SSN, payment info stored directly)
  • Location tracking of individual users
  • Browsing history outside our platform
  • Third-party cookies for advertising

Data Retention

API usage logs are retained for 90 days for operational purposes. Aggregated analytics may be retained indefinitely. Account information is kept until you request deletion.

Data Sharing

We do not sell your personal information. We may share data with:

  • Service providers who assist in operating our platform (under strict confidentiality)
  • Legal authorities when required by law
  • Aggregated, anonymized statistics for research or industry reports

API Data Sources

DC Hub Nexus aggregates publicly available data from sources including:

  • PeeringDB, OpenStreetMap, Wikidata
  • SEC EDGAR filings
  • EIA Energy API, Grid Status API, FCC Broadband Map
  • Public RSS feeds and government data portals

Facility and infrastructure data displayed is sourced from public records and is provided for informational purposes.

Your Rights

You have the right to:

  • Request access to data we hold about you
  • Request correction or deletion of your account data
  • Opt out of non-essential communications
  • Revoke API keys at any time

Security

We implement industry-standard security measures including HTTPS encryption, secure API key hashing, and access controls. However, no system is 100% secure, and we encourage responsible disclosure of any vulnerabilities.
