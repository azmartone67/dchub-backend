FAQ

Can I try before I pay?
Yes. The Free tier gives you 25 queries/day across 5 core tools. Connect via MCP or REST API and explore the data. Upgrade to Developer when you need more endpoints or higher limits.
How does billing work?
Monthly subscription via Stripe. Cancel anytime from your dashboard, no contracts. Your access continues through the end of the billing period.
What happens if I hit my daily limit?
Queries return a friendly limit message with your reset time. No overage charges. If you consistently hit limits, upgrade to Pro for 10x the quota.
Do you support REST API too?
Yes. Every MCP tool is also available as a REST endpoint at dchub.cloud/api/v1/. Same auth, same data, same rate limits.
What AI clients work with the MCP?
Claude Desktop, Claude Code, ChatGPT (via plugin), Cursor, Windsurf, and any client that supports MCP streamable-http transport.
How fresh is the data?
Facility data syncs daily. Grid, fiber, and market data update weekly. Subsea cables and carrier presence refresh weekly from TeleGeography and PeeringDB.
