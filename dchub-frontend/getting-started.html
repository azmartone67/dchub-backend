✅ What Changes When You Upgrade

Your API access jumps from 100 to 10,000 calls/day. You get a new read/write API key, full MCP integration, and priority support. Your dashboard will reflect your new plan.

Using Your API Key

Your API key unlocks programmatic access to DC Hub's data center intelligence.

Making Your First Call

Include your API key in the X-API-Key header with every request:

curl -H "X-API-Key: YOUR_API_KEY" \
  https://dchub.cloud/api/v1/search?q=Ashburn

Search for Facilities

curl -H "X-API-Key: YOUR_API_KEY" \
  https://dchub.cloud/api/v1/facilities?country=US&limit=10

Something Not Right?

⏳ Didn't Get the Welcome Email?

Check your spam/junk folder for an email from alerts@dchub.cloud. If it's not there after 15 minutes, contact us — we'll get your API key to you right away.

📊 Dashboard Still Shows "Free"?

Try logging out and back in. If it still shows Free, reach out to us with the email you used for payment and we'll fix it immediately.

🔑 API Key Not Working?

Make sure you're using the new Pro key from your welcome email (not your old free key). Include it as X-API-Key in the request header. Still having trouble? Contact us.

💳 Charged Twice or Duplicate Payment?

If you see a duplicate charge, email us immediately. We'll cancel the duplicate and issue a refund — usually processed within 3-5 business days.

Frequently Asked Questions

Quick answers to common questions.

What's the difference between Founding Member and Pro?
▼

They have the same features — 10,000 API calls/day, read/write access, full MCP integration, and priority support. The difference is pricing: Founding Members lock in a $99/mo rate as early adopters, while Pro is $199/mo. The Founding Member program is limited and will close once spots are filled.

Can I change my plan later?
▼

Yes. Contact us to upgrade, downgrade, or cancel. We're building self-service plan management — for now, email us and we'll handle it within 24 hours.

How do I cancel my subscription?
▼

Email us at jonathan@dchub.cloud and we'll cancel your subscription immediately. You'll retain access through the end of your current billing period.

I lost my API key. Can I get a new one?
▼

For security, we only store a hashed version of your key — we can't retrieve the original. But we can generate a new one for you. Email us and we'll send a fresh API key within 24 hours. Your old key will be deactivated.

What data does the API provide?
▼

DC Hub tracks 20,000+ data center facilities across 140+ countries. The API provides facility search, market intelligence, M&A transaction data, capacity pipeline analytics, and energy infrastructure data. Visit the API Docs for full endpoint details.

What is MCP access?
▼

MCP (Model Context Protocol) lets AI assistants like ChatGPT, Claude, and Gemini access DC Hub's data directly. With MCP access, you can ask an AI to search for data centers, analyze markets, or pull facility details — and it queries DC Hub in real time. Pro and Founding Members get full MCP integration.

Is my payment information secure?
▼

Absolutely. All payments are processed through Stripe, a PCI Level 1 certified payment processor. DC Hub never sees or stores your credit card information.

Do you offer Enterprise plans?
▼

Yes — for organizations needing custom API limits, bulk data access, dedicated support, or white-label integrations, contact us to discuss Enterprise options tailored to your needs.
