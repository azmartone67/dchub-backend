Claude Desktop

1Open Claude Desktop → Settings → Developer → Edit Config
2Paste the JSON below and save
3Restart Claude Desktop — DC Hub tools appear automatically
{
  "mcpServers": {
    "dchub": {
      "type": "streamable-http",
      "url": "https://dchub.cloud/mcp"
    }
  }
}

Cursor

1Open Cursor → Settings → MCP Servers → Add Server
2Set type to streamable-http and paste the URL
3Or add to .cursor/mcp.json in your project:
{
  "mcpServers": {
    "dchub": {
      "type": "streamable-http",
      "url": "https://dchub.cloud/mcp"
    }
  }
}

VS Code — GitHub Copilot Agent Mode

1Open VS Code → Settings (JSON) or .vscode/mcp.json
2Add the server config below
3Toggle Agent Mode in Copilot Chat to use DC Hub tools
{
  "servers": {
    "dchub": {
      "type": "http",

  }
}
