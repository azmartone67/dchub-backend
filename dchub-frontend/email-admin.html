SMTP Configuration

Configure these environment variables on your server. Supports GoDaddy Workspace or Microsoft 365.

GoDaddy Workspace Email

SMTP_HOST smtpout.secureserver.net
SMTP_PORT 587

Microsoft 365 from GoDaddy

SMTP_HOST smtp.office365.com
SMTP_PORT 587
SMTP_USER your-email@domain.com
SMTP_PASSWORD ••••••••••••
SMTP_FROM_NAME DC Hub
APP_URL https://dchub.cloud

Environment Variables (Replit Secrets)

# GoDaddy Workspace Email
SMTP_HOST=smtpout.secureserver.net
SMTP_PORT=587
SMTP_USER=your-email@yourdomain.com
SMTP_PASSWORD=your-email-password
SMTP_FROM_NAME=DC Hub
SMTP_FROM_EMAIL=your-email@yourdomain.com
APP_URL=https://dchub.cloud

# --- OR for Microsoft 365 from GoDaddy ---
# SMTP_HOST=smtp.office365.com
